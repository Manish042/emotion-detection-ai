⚙️ API v2.0

Emotion Detection API

A powerful REST API to detect human emotions, faces, and expressions from images or live video streams. Integrate in minutes with any language.

Base URL https://api.emotiondetection.ai/v2

🔑 Authentication

All API requests require an API Key passed in the request header.

Header: Authorization: Bearer YOUR_API_KEY

Get your API key from your dashboard after signing in. Keys are prefixed with ed_live_ for production and ed_test_ for sandbox.
Example Header 
Authorization: Bearer ed_live_xxxxxxxxxxxxxxxxxxxxxxxx
Content-Type: application/json

📡 Detect Emotion — Image

POST /detect/emotion Detect emotions from an image URL or base64

Analyzes a face in the provided image and returns detected emotions with confidence scores.

Request Parameters
ParameterTypeDescription
image_url required* string Publicly accessible URL of the image
image_base64 required* string Base64 encoded image (use if no URL)
min_confidence optional float Minimum confidence threshold 0.0–1.0 (default: 0.5)
return_landmarks optional boolean Return facial landmark points (default: false)
Code Example
Python
import requests

url = "https://api.emotiondetection.ai/v2/detect/emotion"
headers = {
    "Authorization": "Bearer ed_live_xxxxxxxxxxxx",
    "Content-Type": "application/json"
}
payload = {
    "image_url": "https://example.com/face.jpg",
    "min_confidence": 0.6,
    "return_landmarks": False
}

response = requests.post(url, json=payload, headers=headers)
print(response.json())
JavaScript
const response = await fetch("https://api.emotiondetection.ai/v2/detect/emotion", {
  method: "POST",
  headers: {
    "Authorization": "Bearer ed_live_xxxxxxxxxxxx",
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    image_url: "https://example.com/face.jpg",
    min_confidence: 0.6,
    return_landmarks: false
  })
});
const data = await response.json();
console.log(data);
cURL
curl -X POST https://api.emotiondetection.ai/v2/detect/emotion \
  -H "Authorization: Bearer ed_live_xxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{"image_url":"https://example.com/face.jpg","min_confidence":0.6}'
